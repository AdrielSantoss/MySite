MyEnglish

O MyEnglish é um sistema utilizado para a prática de conhecimento teório em inglês, ou seja, é um sistema onde o usuário irá treinar o seu inglês, sem conteúdo teórico, 100% focado na prática fazendo com que o usuário reconheça com base em erros e acertos quais são seus pontos fortes e fracos na língua inglesa.

O MyEnglish foi o projeto de conclusão do curso Técnico de informática, realizado na Alcides Maya, e apresentado no 4 Seminário de Técnologia da Informática. O projeto foi avaliado como um dos melhores da noite, sendo muito bem elogiado pelo público, recebendo uma certificação e uma premiação de 50% de desconto na graduação, como recompensa.

Previous Next

O sistema segue uma ideia de design minimalista e User Friendly (amigável para usuários), com telas de design simples, porém atrativas. O sistema contém modo DARK (escuro) e Light (claro), somente o cabeçalho e o rodapé possuem cores fixas. Para o desenvolvimento do design, foi utilizado a biblioteca Styled-componetes, styled-icons (ícones) e códigos CSS3.

Para utilizar o sistema é necessário registrar-se e logar-se com uma conta, caso contrário somente a página Home será acessível. Após logar, o usuário poderá escolher qual será o nível de dificuldade do questionário e quais os tipos de questões, cada questionário possui 30 questões que são exibidas em ordem aleatória.

O sistema possui cinco tipos de questões, são elas: Tradução, Diálogos, Fala, Audição e Tradução com imagens. Cada tipo possui funcionalidades diferentes, isto é, cada uma possui uma programação e configuração única.

Um processo de gameficação simples de pontuação e recordes foi aplicado no sistema, para deixar a experiencia dos usuários mais divertida. Os pontos variam conforme o nível das questões, as questões de níveis fáceis valem 100 pontos, as de níveis médio valem 200 e dificeis 300. Caso o usuário consiga um novo recorde, aparecerá uma notificação informando-o do feito, apenas quando o usuário conclui as 30 questões que o recorde é salvo e armazenado no banco de dados.

Este projeto, foi desenvolvido com React JS no front-end e Node JS no back-end. No front-end destaca-se o uso das principais bibliotecas de React JS, como: React Hooks, React-router e styled-components, utilizados para a construção e renderização dos componentes do front-end. Além disso, outras tecnologias marcam presença, como o Booststrap, Jquery e Axios, que foram utilizados principalmente para auxiliar na responsividade, dinâmica, efeitos, e comunicação com as rotas da API (Axios).

No back-end, temos a presença princpal de Node JS com Express, para a criação de uma API, cuja função será lidar com os dados recebidos do front-end e se comunicar com o banco de dados (MongoDB), neste comunicação está incluso: Inserção de dados, ler de dados, atualizar dados e deletar dados (CRUD). Para a construção da API, foram utilizados inúmeras bibliotecas, tais como: body-parser, consign, crypto, mongodb, nodemon. todas tecnologias utilizadas para adicionar funcionalidades como: configuração da comunicação com o banco de dados, recebimento de dados através do corpo da página, criptografias de senhas, criação de rotas, criação de um servidor (local) e validação de dados.

Saiba mais detalhes sobre o projeto: https://github.com/AdrielSantoss/MyEnglish/

Articles

Este é um sistema de criação e visualização de artigos que podem ser separados por categorias (também criadas pelo usuário). Este sistema foi criado para a prática de conhecimentos em frameworks de Javascript e conexão de banco de dados MongoDB e PostgreSQL.

Previous Next

Este sistema conta com funcionalidades como: Login e Register de contas, Acesso de administrador com token de validação, funcionalidades de adminstrador e criação de categorias e artigos.

O sistema conta com um Dashboard, contendo informações que são: número de usuário, número de categorias e de artigos em todo o sistema. Existe um menu lateral ao lado esquerdo, que apresenta os artigos e categorias em ordem de árvore, cada categoria agrupando seus devidos artigos relacionados ao tema principal da categoria.

Existe uma tela de criação de categorias e artigos, onde os administradores podem criar e cadastrar os mesmos. Para a criação de categorias é necessário informar se ela terá categorias pai ou categorias filha. Para a criação de artigos, o sistema oferece um editor rico, que permite a inserção e diversas formatações de imagens e textos.

Em questão de design, o sistema possui um design simples e minimalista, que foi implementado através de códigos CSS3, com auxilio de Bootstrap e vue-mq para a responsividade em todas as telas de dispositivos.

Para a construção e renderização do sistema, foi utilizado no front-end o framework de Javascript Vue JS, Foram utilizado suas funcionalidades de componentes juntamente com outras bibliotecas como o Vuex, utilizado para a compartilhação de propriedades entre componentes, e vue-editor para a criação de artigos.

Foi desenvolvido uma API, utilizando Node JS, que conta com inúmeras funcionalidades, sendo as principais: Validação de dados, criação e validação de Token de acesso e conexão com bancos de dados

Com a conexao de banco de dados MongoDB e postgreSQL, mongoDB utilizado para armazenar as informações do Dashboard, e PostgreSQL utilizado para a armazenar os usuários, categorias e artigos cadastrados no sitema.

Saiba mais detalhes sobre o projeto em: https://github.com/AdrielSantoss/Projeto-Artigos

Notes JS

Notes JS, é um sistema criado inspirado no programa de notas para Windows chamado Evernote. Porém com o diferencial de ser um sistema de notas na WEB, programado inteiramente com Javascript contendo praticamente todas as funcionalidades do Evernote.

Previous Next

Notes JS, conta com funcionalidades iguais ou semelhantes a do Evernote, que são: Criação de notas, edição de notas com save automático ao digitar, exclusão de notas e Login e Register de conta. Este sistema possui uma finalidade simples, porém com uma programação complexa e organizada.

Para utilizar o Notes JS, é necessário criar uma conta e logar com a mesma no sistema, sendo assim, o usuário terá a permissão a tela de criação de notas.

O usuário pode criar uma nota, e essa nota será listada no menu na esquerda (todas as notas serão listadas no menu), o sistema indicará qual nota está selecionada, ou seja, qual nota esta na tela do usuário pronta para sofrer mudanças.

O sistema possui uma funcionalidade de auto save, isto é, quando o usuário parar de digitar por alguns milésimos de segundos, o sistema salvará as edições automaticamente. Estas notas poderão ser excluídas pelo usuário quando desejar.

Além disso, o sistema foi desenvolvido com um design chamativo e super atrativo. Design que foi feito utilizando as funcionalidades de estilo do SASS.

O sistema foi feito utilizando React JS no front-end para a construção das telas e componentes visuais. Já, no back-end, foi criado uma API com Node JS e Express com uso de funções e funcionalidades do express-generator, juntamente com bibliotecas de auxilio de criação de API, como consign, body-parser, Crypto, express-validator e mongoose para a conexão com o MongoDB.

Saiba mais detalhes sobre o projeto em: https://github.com/AdrielSantoss/Projeto-Artigos

Sobre mim

Desde então, programo quase todos os dias, fazendo inúmeros cursos gratuitos e pagos em diversas plataformas, comecei inicialmente com linguagens como: Python e Java. Porém, não demorou muito para eu me sentir atraído ao mundo da WEB com o uso de Javascript(ES6), PHP7, HTML5 e CSS3.

Comecei no mundo do desenvolvimento WEB também no Curso em Video, em seu curso de HTML5. Fui evoluindo criando sites simples que eu apresentava como trabalho em meu curso de Informática. Com o passar do tempo fui melhorando e aprendendo cada vez mais sobre programação font-end e back-end, criando aplicações cada vez mais robustas e profissionais. Juntamente com a evolução de algumas habilidades, evoluiu dentro de mim meu amor pelos códigos.

Sou apaixonado por programação, isso me motiva a estudar cada vez mais, passo horas praticando, estudando e desenvolvendo, consumo muito conteúdo acadêmico, leio documentações oficiais e assisto e acompanho conteúdo de entreterimento relacionado a tecnologia e programação. como os canais de Youtube:: Código fonte TV , Programador BR, Gabriel Pato, Filipe Dechamps e Universo programado.
